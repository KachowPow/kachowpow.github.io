## 💡 How It Works

### Document Flow:
```
Type Notes → Click Upload → API Request → MongoDB Saves → 
Display in Grid → View/Edit Anytime
```

### Progress Flow:
```
Add Learning Item → Fill Form → API Request → MongoDB Saves → 
Display in Table → Edit/Update Anytime → Analytics Auto-Updates
```

### Analytics Flow:
```
Data in Database → Calculate Stats → Generate Charts → 
Display Insights → Auto-refresh on updates
```
